# Md Shahidul Islam Shabuz — Academic Website

This folder contains a complete, no-build academic website prepared for GitHub Pages.

## Main features

- Mobile-responsive academic design
- Dark/light mode
- Research, publication, teaching, supervision, leadership, award and news sections
- Updated MSc Engineering information
- MDPI *Algorithms* manuscript listed correctly as **under review**
- Print-ready academic CV page
- Existing profile photograph and PDF CV included
- BookCycleBD project report, live-project link and supervision details included
- Dedicated Leadership & Mentorship page with certificate gallery
- Search-engine and social-sharing metadata
- No advertisements, visitor counters, external frameworks or build tools

## Publish it on GitHub Pages

1. Download and unzip the website package.
2. Sign in to GitHub.
3. Open the repository named `shahidul-shabuz.github.io`.
4. Make a backup before replacing the old files:
   - Click **Code → Download ZIP**, or create a backup branch.
5. Delete or replace the old website files.
6. Upload **all files and folders inside this package** to the repository root.
7. Commit the changes to the `main` branch.
8. Open `https://shahidul-shabuz.github.io/` after one or two minutes.
9. Refresh with `Ctrl + F5` if the old design is cached.

No installation or command line is required.

## Important

The downloadable PDF CV is the file you uploaded. The newer information is already included in `cv.html`, which visitors can print or save as PDF. Replace the PDF later with your fully updated CV.
